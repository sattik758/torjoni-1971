# Torjoni – The Eternal Roar

A web project dedicated to the historic speeches of Bangabandhu Sheikh Mujibur Rahman.

## 🌐 Live Website
https://torjoni-1971.web.app

## 📌 Features
- Historical speech collection
- Bengali content and UI
- Firebase Firestore database integration
- Responsive design
- Embedded YouTube speech videos

## 📂 Project Structure
```
Torjoni/
│── index.html
│── speeches.html
│── speech.html
│── css/
│── js/
│── assets/
│── firebase.json
```

## 🔥 Technologies Used
- HTML, CSS, JavaScript
- Firebase (Hosting + Firestore)
- YouTube Embed API

## 🚀 How to Run Locally
1. Clone the repo
2. Open folder
3. Run with Live Server / open index.html

## ⚙️ Firebase Setup
- Create Firebase project
- Enable Firestore
- Add config in `firebase-config.js`
